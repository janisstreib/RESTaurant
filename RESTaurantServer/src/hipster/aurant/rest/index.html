Create a booking

Create a new Booking using a HTTP-PUT request on /booking/ .

Parameter

Parameter Importance Function
begin=<time> Required, begin < end A UNIX-Timestamp representing the start time of the wished booking.
end=<time> Required, end > begin A UNIX-Timestamp representing the end time of the wished booking.

Cancel a new Booking using a HTTP-DELETE request on /booking/ .

Parameter

Parameter Importance Function
id=<id> Required The id of the booking
